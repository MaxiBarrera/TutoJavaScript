Propiedades de objetos de JavaScript

Anterior Proximo

Un objeto es una colección desordenada de propiedades

Las propiedades son la parte más importante de los objetos de JavaScript.

Las propiedades se pueden cambiar, agregar, eliminar y algunas son de solo lectura.

Acceder a las propiedades de JavaScript

La sintaxis para acceder a la propiedad de un objeto es:

// objectName.property
let age = person.age;

o

// objectName["property"]
let age = person.["age"];

o

// objectName[expression]
let age = person.[x];

Ejemplo

person.firstname + " is " + person.age + " years old.";
Check! 

person["firstname"] + " is " + person["age"] + " years old.";
Check! 

let x = "firstname";
let y = "age";
person[x] + " is " + person[y] + " years old.";
Check!

Agregar nuevas propiedades

Puedes agregar nuevas propiedades a un objeto existente simplemente dándole un valor:


Ejemplo

person.nationality = "English";
Check!

Eliminar propiedades

La palabra clave delete elimina una propiedad de un objeto:


Ejemplo

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 50,
  eyeColor: "blue"
};

delete person.age;
Check!

o eliminar persona["edad"];


Ejemplo

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 50,
  eyeColor: "blue"
};

delete person["age"];
Check!

Nota:

La palabra clave delete elimina tanto el valor de la propiedad como la propiedad misma.

Después de la eliminación, la propiedad no se puede utilizar hasta que se vuelva a agregar.

Objetos anidados

Los valores de propiedad de un objeto pueden ser otros objetos:


Ejemplo

myObj = {
  name:"John",
  30,
  myCars: {
    car1:span.verde"Ford",
    car2:span.verde"BMW",
    car3:span.verde"Fiat"
  }
}

Puede acceder a objetos anidados utilizando la notación de puntos o la notación de corchetes:


Ejemplo

myObj.myCars.car2;
Check! 

myObj.myCars["car2"];
Check! 

myObj["myCars"]["car2"];
Check! 

let p1 = "myCars";
let p2 = "car2";
myObj[p1][p2];
Check!
