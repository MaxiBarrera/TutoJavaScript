JavaScript String indexOf()

El método indexOf() devuelve el índice (posición) de la primera aparición de una cadena en una cadena, o devuelve -1 si no se encuentra la cadena:


Ejemplo

let text = "Please locate where 'locate' occurs!";
let index = text.indexOf("locate");
Check!

Nota

JavaScript cuenta las posiciones desde cero.

0 es la primera posición en una cadena, 1 es la segunda, 2 es la tercera, ...

JavaScript String lastIndexOf()

El método lastIndexOf() devuelve el índice de la última aparición de un texto especificado en una cadena:


Ejemplo

let text = "Please locate where 'locate' occurs!";
let index = text.lastIndexOf("locate");
Check!

indexOf() y lastIndexOf(), ambos devuelven -1 si no se encuentra el texto:


Ejemplo

let text = "Please locate where 'locate' occurs!";
let index = text.lastIndexOf("John");
Check!

Ambos métodos aceptan un segundo parámetro como posición inicial para la búsqueda:


Ejemplo

let text = "Please locate where 'locate' occurs!";
let index = text.indexOf("locate", 15);
Check!

Los métodos lastIndexOf() buscan hacia atrás (desde el final hasta el principio), es decir: si el segundo parámetro es 15, la búsqueda comienza en la posición 15 y busca hasta el principio de la cadena.


Ejemplo

let text = "Please locate where 'locate' occurs!";
let index = lastIndexOf("locate", 15);
Check!

JavaScript String search()

El método search() busca una cadena (o una expresión regular) y devuelve la posición de la coincidencia:


Ejemplo

let text = "Please locate where 'locate' occurs!";
text.search("locate");
Check! 

let text = "Please locate where 'locate' occurs!";
text.search(/locate/);
Check!

¿Te diste cuenta?

Los dos métodos, indexOf() y search(), ¿son iguales?

¿Aceptan los mismos argumentos (parámetros) y devuelven el mismo valor?

Los dos métodos NO son iguales. Estas son las diferencias:


  • El método search() no puede tomar un segundo argumento de posición inicial.
  • El método indexOf() no puede tomar valores de búsqueda potentes (expresiones regulares).

Aprenderá más sobre expresiones regulares en un capítulo posterior.

JavaScript String match()

El método match() devuelve una matriz que contiene los resultados de hacer coincidir una cadena con otra cadena (o una expresión regular).


Ejemplos

Realizar una búsqueda de "ain":

let text = "The rain in SPAIN stays mainly in the plain";
text.match("ain");
Check!

Realizar una búsqueda de "ain":

let text = "The rain in SPAIN stays mainly in the plain";
text.match(/ain/);
Check!

Realizar una búsqueda global de "ain":

let text = "The rain in SPAIN stays mainly in the plain";
text.match(/ain/g);
Check!

Realice una búsqueda global sin distinción entre mayúsculas y minúsculas para "ain":

let text = "The rain in SPAIN stays mainly in the plain";
text.match(/ain/gi);
Check!

Nota

Si una expresión regular no incluye el modificador g (búsqueda global), match() devolverá solo la primera coincidencia en la cadena.

JavaScript String matchAll()

El método matchAll() devuelve un iterador que contiene los resultados de hacer coincidir una cadena con otra cadena (o una expresión regular).


Ejemplo

const iterator = text.matchAll("Cats");
Check!

Si el parámetro es una expresión regular, se debe establecer el indicador global (g); de lo contrario, se lanza un TypeError.


Ejemplo

const iterator = text.matchAll(/Cats/g);
Check!

Si desea realizar búsquedas sin distinguir entre mayúsculas y minúsculas, debe configurar el indicador de insensibilidad (i):


Ejemplo

const iterator = text.matchAll(/Cats/gi);
Check!

Notas

matchAll() Es una característica de ES2020.

matchAll() No funciona en Internet Explorer.

JavaScript String includes()

El método includes() devuelve verdadero si una cadena contiene un valor especificado.

De lo contrario, devuelve false.


Ejemplo

Comprueba si una cadena incluye "mundo":

let text = "Hello world, welcome to the universe.";
text.includes("world");
Check!

Comprueba si una cadena incluye "world". Comienza en la posición 12:

let text = "Hello world, welcome to the universe.";
text.includes("world", 12);
Check!

Notes

includes() Es sensible a mayúsculas y minúsculas.

includes() Es una característica de ES6.

includes() No es compatible con Internet Explorer.

JavaScript String startsWith()

El método startsWith() retorna true si una cadena comienza con un valor especificado.

De lo contrario, devuelve false:


Ejemplo

Devuelve verdadero:

let text = "Hello world, welcome to the universe.";
text.startsWith("Hello");
Check!

Devuelve falso:

let text = "Hello world, welcome to the universe.";
text.startsWith("world");
Check!

Se puede especificar una posición de inicio para la búsqueda:


Devuelve falso:

let text = "Hello world, welcome to the universe.";
text.startsWith("world", 5);
Check!

Devuelve verdadero:

let text = "Hello world, welcome to the universe.";
text.startsWith("world", 6);
Check!

Notas

startsWith() Es sensible a mayúsculas y minúsculas.

startsWith() Es una característica de ES6 .

startsWith() No es compatible con Internet Explorer.

JavaScript String endsWith()

El método endsWith() retorna true si una cadena termina con un valor especificado.

De lo contrario, devuelve false:


Ejemplos

Comprueba si una cadena termina con "Doe":

let text = "John Doe";
text.endsWith("Doe");
Check!

Comprueba si los primeros 11 caracteres de una cadena terminan con "mundo":

let text = "John Doe";
text.endsWith("Doe", 11);
Check!

Notas

endsWith() Es sensible a mayúsculas y minúsculas.

endsWith() Es una característica de ES6 .

endsWith() No es compatible con Internet Explorer.
