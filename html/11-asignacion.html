Operadores de asignación de JavaScript

Los operadores de asignación asignan valores a variables de JavaScript.


Operator Example Same As
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y
**= x **= y x = x ** y

Operadores de asignación de turnos


Operator Example Same As
<<= x <<= y x = x << y
>>= x >>= y x = x >> y
>>>= x >>>= y x = x >>> y

Operadores de asignación bit a bit


Operator Example Same As
&= x &= y x = x & y
^= x ^= y x = x ^ y
|= x |= y x = x | y

Operadores de asignación lógica


Operator Example Same As
&&= x &&= y x = x && (x = y)
||= x ||= y x = x || (x = y)
??= x ??= y x = x ?? (x = y)

Nota

Los operadores de asignación lógica son ES2020.

El operador =

El operador de asignación simple asigna un valor a una variable.


Ejemplos de tareas sencillas

let x = 10;
Check!

El operador +=

El operador de asignación de suma agrega un valor a una variable.


Ejemplos de tareas de suma

let x = 10;
x += 5;
Check! 

let text = "Hello"; text += " World";
Check!

El operador -=

El operador de asignación de resta resta un valor de una variable.


Ejemplo de tarea de resta

let x = 10;
x -= 5;
Check!

El operador *=

El operador de asignación de multiplicación multiplica una variable.


Ejemplo de tarea de multiplicación

let x = 10;
x *= 5;
Check!

El operador **=

El operador de asignación de exponenciación eleva una variable a la potencia del operando.


Ejemplo de tarea de exponenciación

let x = 10;
x **= 5;
Check!

El operador /=

El operador de asignación de división divide una variable.


Ejemplo de tarea de división

let x = 10;
x /= 5;
Check!

El operador %=

El operador de asignación de resto asigna un resto a una variable.


Ejemplo de asignación de remanente

let x = 10;
x %= 5;
Check!

El operador <<=

El operador de asignación de desplazamiento a la izquierda desplaza una variable a la izquierda.


Ejemplo de asignación de desplazamiento a la izquierda

let x = 10;
x <<= 5;
Check!

El operador >>=

El operador de asignación de desplazamiento a la derecha desplaza una variable a la derecha (con signo).


Ejemplo de asignación de desplazamiento a la derecha

let x = 10;
x >>= 5;
Check!

El operador >>>=

El operador de asignación de desplazamiento a la derecha sin signo desplaza una variable a la derecha (sin signo).


Ejemplo de asignación de desplazamiento a la derecha sin signo

let x = 10;
x >>>= 5;
Check!

El operador &=

El operador de asignación AND bit a bit realiza una operación AND bit a bit en dos operandos y asigna el resultado a la variable.


Ejemplo de asignación AND bit a bit

let x = 10;
x &= 5;
Check!

El operador |=

El operador de asignación OR bit a bit realiza una operación OR bit a bit en dos operandos y asigna el resultado a la variable.


Ejemplo de asignación OR bit a bit

let x = 10;
x |= 5;
Check!

El operador ^=

El operador de asignación XOR bit a bit realiza una operación XOR bit a bit en dos operandos y asigna el resultado a la variable.


Ejemplo de asignación XOR bit a bit

let x = 10;
x ^= 5;
Check!

El operador &&=

El operador de asignación lógica AND se utiliza entre dos valores.

Si el primer valor es verdadero, se asigna el segundo valor.


Ejemplo de asignación de AND lógico

let x = 10;
x &&= 5;
Check!

El operador &&= es una característica de ES2020.

El operador ||=

El operador de asignación lógica OR se utiliza entre dos valores.

Si el primer valor es falso, se asigna el segundo valor.


Ejemplo de asignación de OR lógico

let x = 10;
x ||= 5;
Check!

El operador ||= es una característica de ES2020.

El operador ??=

El operador de asignación de coalescencia nula se utiliza entre dos valores.

Si el primer valor no está definido o es nulo, se asigna el segundo valor.


Ejemplo de asignación de coalescencia nula

let x = 10;
x ??= 5;
Check!

El operador ??= es una característica de ES2020.
