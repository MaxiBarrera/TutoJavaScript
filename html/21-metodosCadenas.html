JavaScript String Length

La propiedad length devuelve la longitud de una cadena:


Ejemplo

let text = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
let length = text.length;
Check!

Extracción de caracteres de String

Hay 4 métodos para extraer caracteres de string:


  • El método at(position)
  • El método charAt(position)
  • El método charCodeAt(position)
  • Uso del acceso a la propiedad [] como en las matrices

JavaScript String charAt()

El método charAt() devuelve el carácter en un índice (posición) especificado en una cadena:


Ejemplo

let text = "HELLO WORLD";
let char = text.charAt(0);
Check!

JavaScript String charCodeAt()

El método charCodeAt() devuelve el código del carácter en un índice especificado en una cadena:

El método devuelve un código UTF-16 (un entero entre 0 y 65535).


Ejemplo

let text = "HELLO WORLD";
let char = text.charCodeAt(0);
Check!

JavaScript String at()

ES2022 introdujo el método de cadena at():


Ejemplo

Obtener la tercera letra del nombre:

const name = "W3Schools";
let letter = name.at(2);
Check!

Obtener la tercera letra del nombre:

const name = "W3Schools";
let letter = name.at[2];
Check!

El método at() devuelve el carácter en un índice (posición) especificado en una cadena.

El método at() es compatible con todos los navegadores modernos desde marzo de 2022:


Nota

El método at() es una nueva incorporación a JavaScript.

Permite el uso de índices negativos mientras que charAt() no.

Ahora puedes utilizar myString.at(-2) en lugar de charAt(myString.length-2).


Compatibilidad con navegadores

at() es una característica ES2022

JavaScript 2022 (ES2022) es compatible con todos los navegadores modernos desde marzo de 2023:

Property Access [ ]


Ejemplo

let text = "HELLO WORLD";
let char = text[0];
Check!

Nota

El acceso a la propiedad puede ser un poco impredecible:


  • Hace que los strings parezcan arrays (pero no lo son)
  • Si no se encuentra ningún carácter, [ ] devuelve indefinido, mientras que charAt() devuelve una cadena vacía.
  • Es de solo lectura. str[0] = "A" no genera ningún error (¡pero no funciona!)

Ejemplo

let text = "HELLO WORLD";
text[0] = "A";    // Gives no error, but does not work
Check!

Extracción de partes de un string

Hay 3 métodos para extraer una parte de una cadena:

  • slice(start, end)
  • substring(start, end)
  • substr(start, length)

JavaScript String slice()

slice() extrae una parte de una cadena y devuelve la parte extraída en una nueva cadena.

El método toma 2 parámetros: posición inicial y posición final (final no incluido).


Ejemplo

Cortar una porción de una cuerda desde la posición 7 a la posición 13:

let text = "Apple, Banana, Kiwi";
let part = text.slice(7, 13);
Check!

Nota

JavaScript cuenta las posiciones desde cero.

la primera posición es cero.

la segunda posición es 1.


Ejemplo

Si omite el segundo parámetro, el método eliminará el resto de la cadena:


let text = "Apple, Banana, Kiwi";
let part = text.slice(7);
Check!

Si un parámetro es negativo, la posición se cuenta desde el final de la cadena:


let text = "Apple, Banana, Kiwi";
let part = text.slice(-12);
Check!

Este ejemplo corta una porción de una cadena desde la posición -12 a la posición -6:


let text = "Apple, Banana, Kiwi";
let part = text.slice(-12, - 6);
Check!

JavaScript String substring()

substring() es similar a slice().

La diferencia es que los valores inicial y final menores que 0 se tratan como 0 en substring().


Ejemplo

let str = "Apple, Banana, Kiwi";
let part = str.substring(7, 13);
Check!

Si omite el segundo parámetro, substring() se eliminará el resto de la cadena.

JavaScript String substr()

substr() es similar a slice().

La diferencia es que el segundo parámetro especifica la longitud de la parte extraída.


Ejemplo

let str = "Apple, Banana, Kiwi";
let part = str.substr(7, 6);
Check!

Si omite el segundo parámetro, substr() se eliminará el resto de la cadena.


Ejemplo

let str = "Apple, Banana, Kiwi";
let part = str.substr(7);
Check!

Si el primer parámetro es negativo, la posición cuenta desde el final de la cadena.


Ejemplo

let str = "Apple, Banana, Kiwi";
let part = str.substr(-4);
Check!

Una cadena se convierte en minúsculas con toLowerCase():

JavaScript String toUpperCase()


Ejemplo

let text1 = "Hello World!";
let text2 = text1.toUpperCase();
Check!

JavaScript String toLowerCase()


Ejemplo

let text1 = "Hello World!";       // String
let text2 = text1.toLowerCase();  // text2 is text1 converted to lower
Check!

JavaScript String concat()

concat() une dos o más cadenas:


Ejemplo

let text1 = "Hello";
let text2 = "World";
let text3 = text1.concat(" ", text2);
Check!

El método concat() se puede utilizar en lugar del operador más. Estas dos líneas hacen lo mismo:


Ejemplo

text = "Hello" + " " + "World!";
text = "Hello".concat(" ", "World!");

Nota

Todos los metódos de cadena devuelven una nueva cadena. No modifican la cadena original

Formally said:

Las cadenas son inmutables: las cadenas no se pueden cambiar, solo reemplazar.

JavaScript String trim()

El método trim() elimina los espacios en blanco de ambos lados de una cadena:


Ejemplo

let text1 = "      Hello World!      ";
let text2 = text1.trim();
Check!

JavaScript String trimStart()

ECMAScript 2019 agregó el método String trimStart() a JavaScript.

El método trimStart() funciona como trim(), pero elimina los espacios en blanco sólo desde el comienzo de una cadena.


Ejemplo

let text1 = "      Hello World!      ";
let text2 = text1.trimStart();
Check!

JavaScript String trimEnd()

ECMAScript 2019 agregó el método de cadena trimEnd() a JavaScript.

El trimEnd() método funciona como trim(), pero elimina los espacios en blanco sólo del final de una cadena.


Ejemplo

let text1 = "      Hello World!      ";
let text2 = text1.trimEnd();
Check!

JavaScript String Padding

ECMAScript 2017 agregó dos nuevos métodos de cadena a JavaScript: padStart() y padEnd() para admitir el relleno al principio y al final de una cadena.

JavaScript String padStart()

El método padStart() rellena una cadena desde el principio.

Rellena una cadena con otra cadena (varias veces) hasta que alcanza una longitud determinada.


Ejemplo

Rellene una cadena con "0" hasta que alcance la longitud 4:

let text = "5";
let padded = text.padStart(4,"0");
Check!

Rellene una cadena con "x" hasta que alcance la longitud 4:

let text = "5";
let padded = text.padStart(4,"x");
Check!

Nota

El método padStart() es un método de cadena.

Para rellenar un número, primero convierta el número en una cadena.

Vea el ejemplo a continuación.


Ejemplo

let numb = 5;
let text = numb.toString();
let padded = text.padStart(4,"0");
Check!

Compatibilidad con navegadores

padStart() es una característica de ECMAScript 2017.

ES2017 es compatible con todos los navegadores modernos desde septiembre de 2017:

Chrome 58 Edge 15 Firefox 52 Safari 11 Opera 45

padStart() no es compatible con Internet Explorer.

Ejemplo

let padded = text.padEnd(4,"0");
Check!

let padded = text.padEnd(4,"x");
Check!

Para rellenar un número, primero convierta el número en una cadena.

JavaScript String repeat()

El método repeat() devuelve una cadena con un número de copias de una cadena.

El método repeat() devuelve una nueva cadena.

El método repeat() no cambia la cadena original.


Ejemplo

Crear copias de un texto:

let text = "Hello world!";
let result = text.repeat(2);
Check! 

let text = "Hello world!";
let result = text.repeat(4);
Check!

Sintaxsis

string.repeat(count)

Parámetros

Parameter Description
contar Obligatorio. El número de copias que se desean.

Valor de retorno

Tipo Descripción
Cadena Una nueva cadena que contiene las copias.

Compatibilidad con navegadores

repeat() es una característica de ES6 (JavaScript 2015).

ES6 es totalmente compatible con todos los navegadores modernos desde junio de 2017:

Chrome 51 Edge 15 Firefox 54 Safari 10 Opera 38
May 2016 Apr 2017 Jun 2017 Sep 2016 Jun 2016

repeat() no es compatible con Internet Explorer.

Replacing String Content

El método replace() reemplaza un valor especificado con otro valor en una cadena:


Ejemplo

let text = "Please visit Microsoft!";
let newText = text.replace("Microsoft", "W3Schools");
Check!

Nota

El replace() método no cambia la cadena en la que se llama.

El replace() método devuelve una nueva cadena.

El replace() método reemplaza solo la primera coincidencia.

Si desea reemplazar todas las coincidencias, utilice una expresión regular con el indicador /g activado. Vea los ejemplos a continuación.


De forma predeterminada, el método replace() reemplaza solo la primera coincidencia:


Ejemplo

let text = "Please visit Microsoft and Microsoft!";
let newText = text.replace("Microsoft", "W3Schools");
Check!

De forma predeterminada, el método replace() distingue entre mayúsculas y minúsculas. Escribir MICROSOFT (con mayúsculas) no funcionará:


Ejemplo

let text = "Please visit Microsoft and Microsoft!";
let newText = text.replace("MICROSOFT", "W3Schools");
Check!

Para reemplazar la distinción entre mayúsculas y minúsculas, utilice una expresión regular con un indicador /i (sin distinción entre mayúsculas y minúsculas):


Ejemplo

let text = "Please visit Microsoft and Microsoft!";
let newText = text.replace(/MICROSOFT/i, "W3Schools");
Check!

Nota

Las expresiones regulares se escriben sin comillas.


Para reemplazar todas las coincidencias, use una expresión regular con una bandera /g (coincidencia global):


Ejemplo

let text = "Please visit Microsoft and Microsoft!";
let newText = text.replace(/Microsoft/g, "W3Schools");
Check!

JavaScript String ReplaceAll()

En 2021, JavaScript introdujo el método de cadena replaceAll():


Ejemplo

text = text.replaceAll("Cats","Dogs");
text = text.replaceAll("cats","dogs");
Check!

El método replaceAll() le permite especificar una expresión regular en lugar de una cadena a reemplazar.

Si el parámetro es una expresión regular, se debe establecer el indicador global (g); de lo contrario, se lanza un TypeError.


Ejemplo

text = text.replaceAll(/Cats/g,"Dogs");
text = text.replaceAll(/cats/g,"dogs");
Check!

Nota

replaceAll() es una característica de ES2021 .

replaceAll() No funciona en Internet Explorer.

Convertir un string en una array


Si desea trabajar con una cadena como una matriz, puede convertirla en una matriz.

JavaScript String split()

Una cadena se puede convertir en una matriz con el método split():


Ejemplo

text.split(",")    // Split on commas
text.split(" ")    // Split on spaces
text.split("|")    // Split on pipe
Check!

Si se omite el separador, la matriz devuelta contendrá la cadena completa en el índice [0].

Si el separador es "", la matriz devuelta será una matriz de caracteres individuales:


Ejemplo

text.split("")
Check!
