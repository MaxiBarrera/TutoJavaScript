¿Cómo mostrar objetos JavaScript?

Al mostrar un objeto JavaScript se mostrará [object Object].


Ejemplo

// Create an Object
const person = {
  name: "John",
  age: 30,
  city: "New York"
};

document.getElementById("demo").innerHTML = person;
Check!

Algunas soluciones para mostrar objetos JavaScript son:

  • Visualización de las propiedades del objeto por nombre
  • Visualización de las propiedades de un objeto en un bucle
  • Visualización del objeto mediante Object.values()
  • Visualización del objeto mediante JSON.stringify()

Visualización de propiedades de objetos

Las propiedades de un objeto se pueden mostrar como una cadena:


Ejemplo

// Create an Object
const person = {
  name: "John",
  age: 30,
  city: "New York"
};

// Display Properties
document.getElementById("demo").innerHTML =
person.name + "," + person.age + "," + person.city;
Check!

Visualización de propiedades en un bucle

Las propiedades de un objeto se pueden recopilar en un bucle:


Ejemplo

// Create an Object
const person = {
  name: "John",
  age: 30,
  city: "New York"
};

// Build a Text
let text = "";
for (let x in person) {
  text += person[x] + " ";
};

// Display the Text
document.getElementById("demo").innerHTML = text;
Check!

Nota:

Debes utilizar person[x] en el bucle.

person.x no funcionará (porque x es la variable de bucle).

Uso de Object.values()

Object.values() crea una matriz a partir de los valores de propiedad:


Ejemplo

// Create an Object
const person = {
  name: "John",
  age: 30,
  city: "New York"
};

// Create an Array
const myArray = Object.values(person);

// Display the Array
document.getElementById("demo").innerHTML = myArray;
Check!

Uso de Object.entries()

Object.entries() Simplifica el uso de objetos en bucles:


Ejemplo

const fruits = {Bananas:300, Oranges:200, Apples:500};

let text = "";
for (let [fruit, value] of Object.entries(fruits)) {
  text += fruit + ": " + value + "<br>";
}
Check!

Usando JSON.stringify()

Los objetos JavaScript se pueden convertir en una cadena con el método JSON JSON.stringify().

JSON.stringify() Está incluido en JavaScript y es compatible con los principales navegadores.


Nota:

El resultado será una cadena escrita en notación JSON:

{"nombre": "John", "edad": 50", "ciudad": "Nueva York"}


Ejemplo

// Create an Object
const person = {
  name: "John",
  age: 30,
  city: "New York"
};

// Stringify Object
let myString = JSON.stringify(person);

// Display String
document.getElementById("demo").innerHTML = myString;
Check!
