Aritmética de JavaScript

Operadores aritméticos de JavaScript

Los operadores aritméticos realizan operaciones aritméticas con números (literales o variables).


Operator Description
+ Suma
- Sustracción
* Multiplicación
** Exponentiation (ES2016)
/ División
% Módulo (resto)
++ Incremento
-- Decremento

Operaciones aritméticas

Una operación aritmética típica opera sobre dos números.

Los dos números pueden ser literales:


Ejemplo

let x = 100 + 50;
Check!

o variables:


Ejemplo

let x = a + b;
Check!

o expresiones:


Ejemplo

let x = (100 + 50) * a;
Check!

Operadores y operandos

Los números (en una operación aritmética) se llaman operandos.

La operación (a realizar entre los dos operandos) está definida por un operador.


Operando Operador Operando
100 + 50

Añadiendo

El operador de suma (+) suma números:


Ejemplo

let x = 5;
let y = 2;
let z = x + y;
Check!

Restando

El operador de resta (-) resta números.


Ejemplo

let x = 5;
let y = 2;
let z = x - y;
Check!

Multiplicando

El operador de multiplicación (*) multiplica números.


Ejemplo

let x = 5;
let y = 2;
let z = x * y;
Check!

Divisor

El operador de división (/) dicide números.


Ejemplo

let x = 5;
let y = 2;
let z = x / y;
Check!

Resto

El operador de módulo (%) devuelve el resto de la división.


Ejemplo

let x = 5;
let y = 2;
let z = x % y;
Check!

En aritmética, la división de dos números enteros produce un cociente y un resto.

En matemáticas, el resultado de una operación módulo es el resto de una división aritmética.

Incrementando

El operador de incremento (++) incrementa números.


Ejemplo

let x = 5;
x++;
let z = x;
Check!

Decreciente

El operador de decremento (--) decrementa números.


Ejemplo

let x = 5;
x--;
let z = x;
Check!

Exponenciación

El operador de exponenciación (**) eleva el primer operando a la potencia del segundo operando.


Ejemplo

let x = 5;
let z = x ** 2;
Check!

x ** y produce el mismo resultado que Math.pow(x,y):


Ejemplo

let x = 5;
let z = Math.pow(x,2);
Check!

Precedencia del operador

La precedencia del operador describe el orden en que se realizan las operaciones en una expresión aritmética.


Ejemplo

let x = 100 + 50 * 3;
Check!

¿El resultado del ejemplo anterior es el mismo que 150 * 3, o es el mismo que 100 + 150?

¿Se hace primero la suma o la multiplicación?

Al igual que en las matemáticas escolares tradicionales, primero se realiza la multiplicación.

La multiplicación (*) y la división (/) tienen mayor precedencia que la suma (+) y la resta (-).

Y (como en matemáticas escolares) la precedencia se puede cambiar usando paréntesis.

Al utilizar paréntesis, las operaciones dentro de los paréntesis se calculan primero:


Ejemplo

let x = (100 + 50) * 3;
Check!

Cuando muchas operaciones tienen la misma precedencia (como suma y resta o multiplicación y división), se calculan de izquierda a derecha:


Ejemplo

let x = 100 + 50 - 3;
Check! 

let x = 100 / 50 * 3;
Check!
