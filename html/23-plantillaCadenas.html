Plantillas de cadenas

Cadenas de plantillas

Literales de plantilla


El niño amado tiene muchos nombres


Sintaxis de tics inversos

Las cadenas de plantilla utilizan comillas invertidas (``) en lugar de comillas ("") para definir una cadena:


Ejemplo

let text = `Hello World!`;
Check!

Citas dentro de las cadenas

Las cadenas de plantilla permiten comillas simples y dobles dentro de una cadena:


Ejemplo

let text = `He's often called "Johnny"`;
Check!

Strings multilineales

Las cadenas de plantilla permiten cadenas multilínea:


Ejemplo

let text =
`The quick
brown fox
jumps over
the lazy dog`;
Check!

Interpolación

Las cadenas de plantilla proporcionan una forma sencilla de interpolar variables y expresiones en cadenas.

El método se llama interpolación de cadenas.

La sintaxis es:


${...}

Sustituciones de variables

Las cadenas de plantilla permiten variables en cadenas:


Ejemplo

let firstName = "John";
let lastName = "Doe";

let text = `Welcome ${firstName}, ${lastName}!`;
Check!

El reemplazo automático de variables con valores reales se llama interpolación de cadenas.

Sustitución de expresiones

Las cadenas de plantilla permiten expresiones en cadenas:


Ejemplo

let price = 10;
let VAT = 0.25;

let total = `Total: ${(price * (1 + VAT)).toFixed(2)}`;
Check!

El reemplazo automático de expresiones con valores reales se llama interpolación de cadenas.

HTML Templates


Ejemplo

let header = "Template Strings";
let tags = ["template strings", "javascript", "es6"];

let html = `<h2>${header}</h2><ul>`;
for (const x of tags) {
  html += `<li>${x}</li>`;
}

html += `</ul>`;
Check!

Compatibilidad con navegadores

Template Strings es una característica de ES6 (JavaScript 2015).

ES6 es totalmente compatible con todos los navegadores modernos desde junio de 2017:

Chrome 51 Edge 15 Firefox 54 Safari 10 Opera 38
May, 2016 Apr, 2017 Jun, 2017 Sep, 2016 Jun, 2016

Template Strings no son compatibles con Internet Explorer.
